Security

Built for
Cloud Security

Your API keys are encrypted at rest. All traffic is encrypted in transit. Authentication is handled by Google SSO with JWT session management.

JWT Authentication

Google OAuth sign-in with secure httpOnly JWT cookies. SameSite policies enforced. Tokens auto-expire after 7 days with forced re-authentication.

Encrypted Key Storage

API keys are encrypted with AES-256 before storage. They are never logged, never transmitted in plaintext, and only decrypted in memory at session spawn time.

HTTPS & WSS Only

All traffic runs over TLS. The terminal WebSocket connection uses WSS (WebSocket Secure). No unencrypted fallbacks accepted.

Rate Limiting & Isolation

100 messages/sec per WebSocket connection. Per-user session isolation ensures your PTY processes are fully sandboxed from other subscribers.
